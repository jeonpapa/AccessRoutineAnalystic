# HIRA Pipeline Tracker — 데이터 관리 구조

> **운영 목적**: 심사평가원(HIRA)의 두 핵심 위원회 — 중증(암)질환심의위원회(암질심)와 약제급여평가위원회(약평위) — 의 심의결과를 1차 권위 source(HIRA 공식 보도자료) 기반으로 추적하고, 매체 보도는 통과·미통과 사유 해석에만 활용.

## 데이터 권위 위계

| 권위 단계 | source | 용도 |
|---|---|---|
| **1차 권위** | HIRA 공식 보도자료 (https://www.hira.or.kr/bbsDummy.do?pgmid=HIRAA020041000100) | 안건 약물 list·통과/미통과 결정 사실 확정 |
| 2차 권위 | 보건복지부 고시·건정심 의결 | NHIS 협상 후 등재 사실 |
| 3차 권위 | 한국 의약 전문지 보도 | "왜 통과/미통과인지" 사유 해석, 회사·심평원 협의 동향 |

**원칙**: 약물·차수·결정 사실은 HIRA 공식이 단독 권위. 매체 보도는 사유·맥락 분석에만 사용.

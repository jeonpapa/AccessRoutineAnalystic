## 1. Executive Summary

**4차 차수는 임상적 OS 진성성이 결과를 가른 회차였다.** 처리 5건 중 실적된 2건(베이지노스·엘라히어)은 모두 (a) 차수 직전 OS 데이터 추가 업데이트 또는 (b) 정부 측 D-14 시점 공식 발언이라는 두 조건 중 최소 하나를 충족했고, 미실적된 3건(림카토·알렌시·키스칼리)은 OS 부재·BIA 부담·직접 비교 우위 약물 부재 중 하나에 발목 잡혔다. 

큐 후순위 약물 9개는 모두 QUEUE_PENDING 상태로 잔류했고, 그중 **웰리렉(MSD, 3/20 신청)은 4/15·5/27 두 차수 연속 미실적**으로 본인 앞 큐 잔류 약물이 최소 10건 이상으로 추정된다. 7/8 5차에서 처리 가능성이 매체 신호 기준 가장 높지만, 자동 보장은 불가하며 6/24 D-14 시점에 재차 모니터링 필요.

본 회차에서 학습된 패턴 중 **PATTERN-002(정부 발언 D-14 타이밍)**·**PATTERN-003(N차 도전 + OS 업데이트)**은 이미 코드화 가능한 강한 시그널이며, KR-RULE-031 신설 후보 1건이 도출됐다(§7).

## 3. 사전 신호 vs 실제 — Confusion Matrix

**예측 정의 A** (Tier A 사전 leak ∪ Tier D 5/14 거명, broad): 11개 약물 예측 ON_AGENDA

|  | 실제 ON_AGENDA | 실제 QUEUED |
|---|:-:|:-:|
| 예측 ON_AGENDA | **5 (TP)** — 림카토·엘라히어·알렌시·베이지노스·키스칼리 | **6 (FP)** — 알트라피·엔허투·포튜시아·보라네고·웰리렉·폴리비 |
| 예측 QUEUED | **0 (FN)** | **3 (TN)** — 티루캡·장핀이로·간타스주 |

→ Precision **0.45** / Recall **1.00** / F1 **0.62**

**예측 정의 B** (메디칼타임즈 5/15 D-12 leak only, strict): 6개 약물 예측 ON_AGENDA

|  | 실제 ON_AGENDA | 실제 QUEUED |
|---|:-:|:-:|
| 예측 ON_AGENDA | **3 (TP)** — 엘라히어·알렌시·베이지노스 | **3 (FP)** — 알트라피·포튜시아·엔허투 |
| 예측 QUEUED | **2 (FN)** — 림카토·키스칼리 | **6 (TN)** |

→ Precision **0.50** / Recall **0.60** / F1 **0.55**

**해석**:
- **Recall 1.00 (정의 A)**: 실제 5건 안건 모두 사전 신호 ≥1 잡힘 — 본 14매체 커버리지가 안건 발견 자체는 완벽
- **Precision 낮음**: 큐 inventory(메디칼옵저버 5/27)와 안건 leak(메디칼타임즈 5/15)을 동일 신호로 취급한 결과. 두 매체의 정보 본질을 구분하는 KR-RULE-030 적용 시 precision 개선 가능
- **림카토 FN(정의 B)**: 메디칼타임즈 D-12 누락된 안건이 큐로셀 IR(머니투데이 thebio 5/14)로만 거명됨 — **단일 매체 의존, [CROSSREF-MISSING] 위배**

## 4. 큐 처리 분기점 분석

### 4.1 실적 2건 공통 분기점

| 분기점 | 베이지노스 | 엘라히어 |
|---|---|---|
| OS 진성성 (PATTERN-001) | ESMO 2025 OS 데이터 4월 추가 | ✗ (단독 OS 우월) |
| 정부 D-14 발언 (PATTERN-002) | ✗ | 사무관 5/12 토론회 "5월 안으로 적극 검토" |
| N차 도전 + OS (PATTERN-003) | **4차 도전 + OS 업데이트** | 1차(첫 신청) |

→ **분기점**: OS 데이터 강도(베이지노스) OR 정부 측 명시적 D-14 발언(엘라히어). 둘 중 하나만 충족해도 실적.

### 4.2 미실적 3건 공통 분기점

| 분기점 | 림카토 | 알렌시 | 키스칼리 |
|---|---|---|---|
| 임상 약점 | 장기 OS 부재, RSA 구조 미확정 | adjuvant 장기 투여 BIA 부담 | NATALEE vs 베이지노스 monarchE 7년 OS 직접 비교 열위 |
| 정부 신호 | ✗ | ✗ | ✗ |
| 카테고리 압박 | CAR-T 첫 국산(레퍼런스 부재) | NSCLC adjuvant 신규 인디케이션 | 동일 차수 베이지노스와 직접 비교 |

→ **분기점**: 동일 클래스 약물이 동일 차수에 처리될 때 OS 우위 약물만 실적 (키스칼리 vs 베이지노스). 학습 패턴 PATTERN-003 변형 신호.

## 5. MSD 항암 자산 영향 분석

### 5.1 웰리렉 (Welireg / belzutifan / HIF2A)

| 지표 | 값 |
|---|---|
| 신청일 | 2026-03-20 |
| 시도 차수 | 1차 (4차 차수 시점) |
| 현재 상태 | `QUEUE_PENDING` |
| 4/15 3차 처리 | ✗ |
| 5/27 4차 처리 | ✗ |
| 본인 앞 큐 잔류 추정 | ≥10건 (capacity 5건 × 2차수 미처리) |
| 7/8 5차 사전 신호 | 메디칼옵저버 5/27 거명 ✓ · Tier D 5/14 거명 ✓ · Tier G(5만 청원 + 국회 보건복지위 자료) ✓ |

**핵심 판단**: 신청 3/20 후 2차수 연속 큐 잔류는 본인 앞 적체가 깊다는 의미. 단순 FIFO로는 7/8 5차에 도달 가능하지만, 정부 측 D-14 발언이 없으면 6차로 밀릴 수도 있음. **6/24 D-14 시점에서 정부·KOL 신호 재모니터링 필수** — 본 약물이 6/24 검증 1순위 타깃.

### 5.2 키트루다 (Keytruda / pembrolizumab / PD-1)

본 4차 차수는 키트루다 안건 없음. 키트루다는 이미 다회 등재된 약물로 별도 트랙(개별 적응증 확대 신청)에서 운영 추정. **확인 사항(§7-3)**: 키트루다 신규 적응증 확대 신청이 암질심 큐에 있는지, 별도 트랙인지 사용자 확인 필요.

### 5.3 경쟁 클래스 (PD-1 / CDK4-6 / CAR-T)

| 클래스 | 본 차수 영향 |
|---|---|
| CAR-T | 림카토 미실적 → **국산 1호 CAR-T 등재 첫 좌초**. 후속 CAR-T(킴리아 신규 적응증 등) 등재 표준 정립 지연 우려 |

## 7. 사용자 검토 요청 사항 (KPAD KB 패치 후보)

### 7-1. KR-RULE-031 신설 후보 — "동일 클래스 동일 차수 비교 우위 약물만 실적" 

**근거**: 키스칼리(NATALEE) vs 베이지노스(monarchE) 동일 차수 처리, 베이지노스만 APPROVED. 동일 패턴이 7/8 5차에 재현될 가능성(예: 동일 차수에 HER2 저발현 표적 약물 2종 동시 상정 시).
**조치**: PATTERN-007 후보로 등록. 다음 차수 검증 후 확정.

### 7-2. KR-RULE-030 강화 — "메디칼옵저버 inventory ≠ 안건 leak"

**근거**: 본 차수 confusion matrix에서 두 매체를 동일 신호로 취급 시 Precision 0.45로 떨어짐. KR-RULE-030 §How to apply에 "메디칼옵저버 5/27 inventory 거명만으로 ON_AGENDA 예측 금지" 명시 추가 필요.

### 7-3. 키트루다 트랙 별도 운영 확인 (사용자 답변 필요)

키트루다 신규 적응증 확대 신청이 (a) 암질심 큐에 등록되어 있는지, (b) 별도 보건복지부 직접 트랙으로 운영되는지 확인 필요. 본 답변에 따라 `amjilsim_drugs` 시드에 키트루다 적응증별 row 분리 여부 결정.

### 7-4. 차수 numbering 확정 — §2.3 vs §6.1

5/27=4차 확정(본 보고서 기준). 7/8=5차로 통일. 핸드오프 문서 §6.1 표(7/8=6차)는 오기로 처리, calendar.py·DB seed note 정정 batch fix 예정.
